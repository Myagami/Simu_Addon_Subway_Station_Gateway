□地下鉄駅 出入口セット Version1(正式版)

●概要
今回は当アドオンをダウンロードしていただき、まことにありがとうございます。
このアドオンは地下に建設されるから目立たない地下鉄駅の出入り口を用意し、そこに地下線が走っていることを目立たせる事を目的に制作したアドオンです。
様々なバリエーションをご用意致しましたので使えるかと思います

●収録内容
・Subway_Station_Gateway_Box
1x1で立てられる箱型のシンプルな出入り口、4方向から出入りできるようになっています

・Subway_Station_Gateway_Single_S
1x1で地下鉄出入口定番の階段の出入り口だけバージョン

・Subway_Station_Gateway_Single_S-EV1
・Subway_Station_Gateway_Single_S-EV2
バリアフリーに考慮してエレベーターを付けたバージョン

・Subway_Station_Gateway_H1
・Subway_Station_Gateway_H2

・Subway_Station_Gateway_W1
・Subway_Station_Gateway_W2
券売機などの設備を地上に持ってきたバージョンです、それぞれ1x2と2x1で立てられます


・Subway_Station_Gateway_TimeTower
街の小さな時計台をイメージした出入口です

●2次利用について
最低限な話として「転載行為は禁止します」
アドオン制作の元としては自由に扱ってください、その際に作者であるM.Yagamiの名前を入れていただけると幸いです。
